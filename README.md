### Labyrinth Game

##  Описание

**Labyrinth Game** — это консольная текстовая игра, где вы оказываетесь в таинственном лабиринте, полном загадок, ловушек и спрятанных предметов.  
Ваша цель — исследовать все комнаты, найти ключ и открыть сундук с сокровищем, чтобы выбраться победителем.

## ⚙️ Установка и запуск

### 1. Склонировать проектbash

git clone https://github.com/your-username/project1_Shevchenko_Sergey.git

cd project1_Shevchenko_Sergey


### 1. Склонировать проект
poetry install
### 3. Запустить игру
poetry run project

#### 4. Правила игры
После запуска вы окажетесь в входе лабиринта.

В каждой комнате можно:

Осмотреться (look) осмотреться 
Взять предмет (take <название>)
Использовать предмет (use <название>)
Решить загадку (solve)
Перемещаться между комнатами (go north, south, east, west или просто north, south и т.д.)
Проверить свой инвентарь (inventory)
Получить подсказку (help)
Выйти из игры (quit)

##### 5. Цель игры

Чтобы выиграть:
Исследуйте лабиринт, собирайте предметы.
Решайте загадки — правильные ответы могут открывать новые возможности.
Найдите ключ и используйте его, чтобы открыть сундук с сокровищем.
После открытия сундука игра заканчивается победой.
Но будьте осторожны — в некоторых комнатах спрятаны ловушки и случайные события!
